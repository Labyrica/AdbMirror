# Plan 03-01: MainWindow Shell and Layout

**Phase**: 03-mirror-ui
**Depends on**: Phase 2 complete
**Requirements**: UX-03 (simplified UI)

## Objective

Create the MainWindow shell with two-column layout matching WPF reference. Left column for status/logs, right column for controls. Establish the visual hierarchy with header, content areas, and proper spacing using FluentAvaloniaUI.

## Reference

WPF MainWindow.xaml uses:
- 760x460 window, 2.2:1.4 column ratio (left:right)
- Left: status panel with header, status line, ADB info, event log
- Right: device summary, Mirror button (hero), quality selector, settings, credits
- LabCardBorder style for content panels
- 24px outer margins, 16px inner gaps
